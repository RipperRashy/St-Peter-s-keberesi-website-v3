# St. Peter's Keberesi Website — Image Guide
=============================================

## HOW TO REPLACE IMAGES
All images go inside the `images/` folder.
To swap any image, just replace the file in the `images/` folder with your school photo
and keep the SAME filename. That's it!

## IMAGE SLOTS & FILENAMES
-----------------------------------------------

### HERO / BANNER PHOTOS (appear at the top of each page)
- images/hero-home.jpg        → Home page banner
- images/hero-about.jpg       → About Us page banner
- images/hero-academics.jpg   → Academics page banner
- images/hero-admissions.jpg  → Admissions page banner
- images/hero-student-life.jpg→ Student Life page banner
- images/hero-sports.jpg      → Sports page banner
- images/hero-news.jpg        → News & Events page banner
- images/hero-contact.jpg     → Contact page banner

### HOME PAGE
- images/principal.jpg        → Principal's photo (home page)
- images/school-building.jpg  → School building photo (home page about section)

### GALLERY (gallery section on home page)
- images/gallery-1.jpg        → Gallery photo 1
- images/gallery-2.jpg        → Gallery photo 2
- images/gallery-3.jpg        → Gallery photo 3
- images/gallery-4.jpg        → Gallery photo 4
- images/gallery-5.jpg        → Gallery photo 5
- images/gallery-6.jpg        → Gallery photo 6

## RECOMMENDED IMAGE SIZES
- Hero/banner photos: 1400 x 500 pixels (wide landscape)
- Principal photo: 400 x 500 pixels (portrait)
- Gallery photos: 600 x 450 pixels (landscape)
- School building: 800 x 500 pixels

## TIPS
- Use JPG format for photos (smaller file size)
- Keep file sizes under 500KB for fast loading
- You can use free tools like https://squoosh.app to compress photos
- If a photo looks stretched, make sure it matches the recommended size above
